# iTunesController

Unityエディタで再生・停止したときに、iTunesも再生・停止するやつ
Macのみ（Mac OS は Sierra で確認）

## Requirements

- Unity v2017.2+

## Installation

[ここ](https://anz-note.ghost.io)から**iTunesController.package.zip**をダウンロードしてから展開後、Unityでインポート
